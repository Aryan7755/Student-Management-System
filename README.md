Student Management System

A console-based Student Management System developed in Java. This project allows users to manage student records, update academic performance, and analyze results using sorting and searching techniques.

Features
-Add student records (ID, name, marks)
-Update student marks with validation
-Search students by ID or name
-Sort students by marks (descending order)
-Display top-performing students (Top N)
-View all student records

Concepts Used
-Object-Oriented Programming (OOP)
-Implementation of classes, objects, and encapsulation
-Collections Framework
-Use of ArrayList for dynamic data storage
-Sorting
-Implemented using Comparator and Collections.sort()
-Searching
-Linear search for retrieving student records
-Input Handling
-CLI-based interaction using Scanner

Tech Stack
Language: Java
Environment: IntelliJ IDEA (or any Java IDE)
Type: Console-based application

How to Run
-Clone the repository:
-git clone <repository-url>
-Open the project in your preferred IDE
-Navigate to:src/MainApp.java
-Run the application

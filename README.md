Student Management System

A console-based Student Management System developed in Java. This project allows users to manage student records, update academic performance, and analyze results using sorting and searching techniques.

-> Features
    1.Add student records (ID, name, marks)
    2.Update student marks with validation
    3.Search students by ID or name
    4.Sort students by marks (descending order)
    5.Display top-performing students (Top N)
    6.View all student records
    
-> Concepts Used
    1.Object-Oriented Programming (OOP)
    2.Implementation of classes, objects, and encapsulation
    3.Collections Framework
    4.Use of ArrayList for dynamic data storage
    5.Sorting
    6.Implemented using Comparator and Collections.sort()
    7.Searching
    8.Linear search for retrieving student records
    9.Input Handling
    10.CLI-based interaction using Scanner
    
-> Tech Stack
    Language: Java
    Environment: IntelliJ IDEA (or any Java IDE)
    Type: Console-based application

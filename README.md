# Student Management System

A console-based Student Management System developed in Java. This project allows users to manage student records, update academic performance, and analyze results using sorting and searching techniques.

## Features
- **Add student records:** ID, name, marks
- **Update student marks:** Includes validation logic
- **Search students:** Search by ID or name
- **Sort students:** Sort by marks in descending order
- **Top Performers:** Display the Top N students
- **View All:** Display all student records

## Concepts Used
- **OOP:** Implementation of classes, objects, and encapsulation
- **Collections Framework:** Use of `ArrayList` for dynamic data storage
- **Sorting:** Implemented using `Comparator` and `Collections.sort()`
- **Searching:** Linear search for retrieving records
- **Input Handling:** CLI-based interaction using `Scanner`

## Tech Stack
- **Language:** Java
- **Environment:** IntelliJ IDEA (or any Java IDE)
- **Type:** Console-based application
